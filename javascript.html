Vault

find

The `find``` method takes one argument, the id of the vault. It returns the vault object with that id.

data = vault.find(3)

create

The `create``` method takes two arguments, the data's container and the data that you want to save.

vault.create('superheros', {'name': 'Batman'}

update

The `update``` method takes two arguments, the id of the vault and the data to update the vault with.

data = vault.find(3)
data.name = 'Superman'
vault.update(3, data)

count

The `count``` method takes one argument, the data's container and returns the number of items that have a container of the same name.

count = vault.count('superheros')

Push

sendToToken

The `sendToToken``` method takes four arguments:

  • The app_id of the device you want to send to.
  • The token of the device you want to send to.
  • The message you want to send
  • A boolean value

* `true``` if you want use the development certificate

* `false``` if you want use the production certificate. This is the default.

sendToAlias

  • The app_id of the device you want to send to.
  • The alias of the device you want to send to.
  • The message you want to send
  • A boolean value

* `true``` if you want use the development certificate

* `false``` if you want use the production certificate. This is the default.

sendToTags

  • The app_id of the device you want to send to.
  • An array of tags of the device you want to send to.
  • The message you want to send
  • A boolean value

* `true``` if you want use the development certificate

* `false``` if you want use the production certificate. This is the default.
